## pacstrap

Run pacstrap on the chroot /mnt after setting ParallelDownloads in pacman.conf. The following packages are installed:

- base
- linux
- linux-firmware
- networkmanager
- stow
- git
- grub
- sudo
- openssh
- rsync
- wget
- vim
- make
- gcc
- pkg-config
- fakeroot
- lvm2
- python

### Variables

- parallel_downloads
