## create_users

Add the system users. All users and passwords should be vaulted in host_vars/your_host/vault.yml

### Variables

vaulted in host_vars/your_host/vault.yml):
- admin_user (uid 1000)
- ancillary user (uid 1001)
- build_user (uid 1002)
- admin_pass
- ancillary_user_pass
- build_user_pass
